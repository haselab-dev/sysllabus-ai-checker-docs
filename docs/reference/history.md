---
title: 履歴画面
description: 過去の実行履歴を確認し，再読み込みや保存場所の確認を行う画面
---

履歴画面では，これまでに実行したチェックの状態や結果を一覧で確認できます．途中で止まった処理を再開したり，実行結果のファイルを再ダウンロードしたりできます．

## 表示内容

- 実行日時: そのチェックを実行した日時です．
- Run ID: 各実行に付与される識別子です．
- 状態: `running`，`paused`，`completed`，`failed`，`cancelled`，`interrupted` などの状態が表示されます．
- 進捗: 処理済みファイル数，処理済み行数，進捗率を確認できます．
- ファイル一覧: その実行に含まれる各 CSV の状態と進み具合を確認できます．

## 途中で止まった実行を再開する場合

`Load` を押してチェック画面へ移動し，その後 `再開` を押します．

## 実行結果を再ダウンロードする場合

`Load` を押してチェック画面へ移動し，その後 `実行結果をZIPでダウンロード` を押します．

チェックに使う入力 CSV の条件と注意点です．

## 追加できるファイル

- 使えるのは `.csv` ファイルだけです．
- 複数ファイルをまとめて追加できます．
- 同じファイルは 2 回追加できません．
- 別フォルダにある同名ファイルは同時に追加できます．画面では `sample_1.csv`，`sample_2.csv` のように表示されます．

## 列名のルール

- 同じ列名があると，2 列目以降は `列名_1`，`列名_2` のように読み替えられます．プロンプトの設定での`調査対象の列名`でもその名前を使ってください．
- 列名に改行がある CSV も読めますが，prompt設定側で指定しづらいため非推奨です．

## 文字コードについて

- 設定画面の `文字コード` で指定した値で読み取ります．
- Windows では `shift_jis`，それ以外の OS では `utf-8` で作られることが一般的です．文字コードが分からなければ，`auto` でも大丈夫です．

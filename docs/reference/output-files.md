チェック結果として作られるファイルの説明です．

## 保存されるファイル

出力ファイルは ZIP ファイルです．中を見るには解凍が必要です．解凍すると以下のファイルが作られます．

- `result.csv`: 元の列に `<項目名>_結果` と `<項目名>_理由` を追加した CSV です．
- `result-binary.csv`: `〇` / `×` を `1` / `0` にした CSV です．`No`，`授業コード`，`授業科目名` も入ります．
- `result.md`: 判定結果を Markdown でまとめたファイルです．
- 複数ファイルを実行したときは，履歴フォルダ内でファイルごとに `1-<1つ目のファイル名>`，`2-<2つ目のファイル名>` のような別フォルダが作られます．

## 文字コードについて

- 出力 CSV の文字コードは，入力 CSV と同じです．

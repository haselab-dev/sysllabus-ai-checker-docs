## このアプリでできること

- シラバスの CSV を読み込み，内容をまとめて確認できる
- LLM を使って，不足や表現の問題がないかを効率よくチェックできる
- レビュー担当者ごとの差を抑えながら，確認作業を進めやすくできる
- プロンプトを変更して，運用に合わせた観点でチェックできる
